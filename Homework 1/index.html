Base

  • git config --global user.name blabla yказываем имя пользователя
  • git config --global user.email blabal@bla указываем email
  • git init создает новый репозиторий Git. Команда git init в текущем рабочем каталоге создает подкаталог .git со всеми необходимыми метаданными Git для нового репозитория.
  • git status просмотр текущих изменений
  • git add добавить файл в локальный репозиторий, запись изменений в репозиторий для последующего коммита
  • git commit берёт все данные, добавленные в индекс с помощью git add, и сохраняет их слепок во внутренней базе данных
  • git log выводит список коммитов созданных в данном репозитории в обратном хронологическом порядке.
  • git checkout позволяет перемещаться между ветками, созданными командой git branch
  • git branch посмотреть в какой ветке мы находимся
  • git push публикация выгружаемых локальных изменений в центральном репозитории
  • git pull скачивается актуальная версия удаленного репозитория и все изменения применяются к локальному репозиторию
  • git merge объединяет несколько последовательностей коммитов в общую историю, используется для объединения двух веток
  • git clone используется для клонирования репозитория, скачивание репозитория
.gitignore файл это простой текстовый файл, в каждой строке которого содержится шаблон файла или каталога, который необходимо проигнорировать
